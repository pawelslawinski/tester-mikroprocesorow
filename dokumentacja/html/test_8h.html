enum StanTestu

Określa dostępne stany przeprowadzanego testu.

Wartości wyliczeń
brak_danych 

Jeszcze nie został wybrany żaden test do przeprowadzenia.

wybrany 

Test został już wybrany.

rozpoczety 

Test właśnie się rozpoczął

w_trakcie 

Test jest wykonywany.

Test został zakończony.

bledny 

Podczas testu wystąpił błąd.

◆ OdczytajTypTestu()

void OdczytajTypTestu ( )

◆ Testuj()

void Testuj ( )

Mikroprocesor jest testowany wybranym wcześniej testem.

◆ TestujADC()

int TestujADC ( )

Mikroprocesor programowany jest odpowiednim programem testującym przetwornik A/C. Sygnał pobudzający generowany jest przez ustawiony generator sygnałem sinusoidalnym. Na końcu sprawdzana jest poprawnośc otrzymanych wyników.

Zwraca
1 jeśli test zakończył się sukcesem, 0 jeśli wystąpił błąd.

◆ TestujPortyIN()

int TestujPortyIN ( )

Mikroprocesor programowany jest odpowiednim programem testującym sprawnośc portów wejściowych. Prostokątny sygnał testujący generowany jest przez generator. Na końcu sprawdzana jest poprawnośc otrzymanych wyników.

Zwraca
1 jeśli test zakończył się sukcesem, 0 jeśli wystąpił błąd.

◆ TestujPortyOUT()

int TestujPortyOUT ( )

Mikroprocesor programowany jest odpowiednim programem testującym sprawnośc portów wyjściowych. Sygnały generowane przez mikroprocesor są odczytywane przez odpowiednie czujniki. Na końcu sprawdzana jest poprawnośc otrzymanych wyników.

Zwraca
1 jeśli test zakończył się sukcesem, 0 jeśli wystąpił błąd.

◆ TestujTimer()

int TestujTimer ( int  numerTimera)

Mikroprocesor programowany jest odpowiednim programem testującym sprawnośc układów czasowych. Na końcu sprawdzana jest poprawnośc działania mikroprocesora.

Parametry
numerTimeranumer testowanego Timera.
Zwraca
1 jeśli test zakończył się sukcesem, 0 jeśli wystąpił błąd.
