int TestujADC ( )

Mikroprocesor programowany jest odpowiednim programem testującym przetwornik A/C. Sygnał pobudzający generowany jest przez ustawiony generator sygnałem sinusoidalnym. Na końcu sprawdzana jest poprawnośc otrzymanych wyników.

◆ TestujPortyOUT()

1 jeśli test zakończył się sukcesem, 0 jeśli wystąpił błąd.
